# Docker & Containerization

This project focuses on understanding containerization concepts using Docker.

## Project Overview

The objective of this project was to learn how applications can be containerized, deployed, and isolated using Docker.

The repository includes a custom Dockerfile used to build a containerized environment for application deployment and testing.

## Key Concepts Practiced

- Containerization principles
- Docker image creation
- Writing Dockerfiles
- Layered image architecture
- Environment isolation
- Reproducible development environments

## Technologies Used

- Docker
- Linux
- Container-based deployment

## Learning Outcomes

- Understanding how containers differ from virtual machines
- Building portable application environments
- Deploying applications consistently across systems
- Basic DevSecOps awareness
